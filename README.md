# Retail Customer Behavior & Sentiment Analysis

## About the Project

This project is about analyzing customer shopping behavior and product reviews in a retail environment. The main idea is to understand how customers purchase, what they prefer, and how satisfied they are based on their reviews.

We used a combination of tools like Python, MySQL, and Power BI to build a complete data analytics pipeline. The goal is to take raw data and convert it into useful insights that can help businesses make better decisions.

---

## Tools Used

* Python – for data cleaning and analysis
* MySQL – for storing data and running queries
* Power BI – for creating dashboards and visualizations

---

## What We Did

* Cleaned and prepared the dataset using Python
* Connected Python to MySQL and stored the data
* Performed SQL queries to analyze business insights
* Created visualizations using Python
* Built interactive dashboards in Power BI
* Performed sentiment analysis on customer reviews

---

## Key Insights

* Clothing is the top revenue-generating category
* Customers prefer digital payment methods like PayPal and cards
* Discounts and subscriptions do not significantly affect spending
* Customer spending is almost the same across all age groups
* Most reviews are neutral to positive
* Flipkart has more customer review activity compared to Amazon
